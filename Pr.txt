Отчёт по практике №12 "GIT" студентки группы БИК2301 Караевой Екатерины
1) Скачиваем GIT на компьютер
2) Переходим в созданную директорию
3) Cоздаём в ней локальный репозиторий с расширением ".git" с помощью команды "git init"
4) Создаём текстовый файл с отчётом в выбранной директории и сохраняем изменения с помощью команды "git commit"
5) Добавляем новую ветку "Pr2" с помощью команды "git branch" и "git checkout"
6) Добавляем отчёт в главную ветку
7) Заводим аккаунт на github и создаём репозиторию "homework-IIT"
8) Создаём удалённый репозиторий командой "git remote add "Practica"" со ссылкой на репозиторию "homework-IIT"
9) Добавляем главную ветку в "Practica" с помощью команды "git push Practica master"
10) В главную ветку удалённого репозитория загружаем все исходники нашего сайта и запускаем его через вкладку "Pages"
